🚀 PromptNext

PromptNext is an all-in-one developer productivity platform designed to help developers improve their workflow using curated AI-assisted prompts. The platform provides prompts for resume enhancement, code reviews, and technical guidance, enabling users to generate better outputs and streamline repetitive tasks.

✨ Features

- 📝 Resume Enhancement Prompts
  
  - ATS-friendly resume suggestions
  - Professional summary generation
  - Resume improvement guidance

- 💻 Code Review Assistance
  
  - Code optimization prompts
  - Best practices recommendations
  - Debugging and refactoring suggestions

- 🔐 Secure Authentication
  
  - User sign-up and login using Firebase Authentication
  - Protected routes for authenticated users

- 📱 Responsive Design
  
  - Mobile-first user interface
  - Optimized for desktop, tablet, and mobile devices

- ♻️ Reusable Components
  
  - Modular component architecture
  - Easy scalability and maintenance

🚀 Getting Started

Clone the Repository

git clone https://github.com/yourusername/promptnext.git

Navigate to the Project

cd promptnext

Install Dependencies

npm install

Run the Development Server

npm run dev

Open "http://localhost:5173" in your browser.
